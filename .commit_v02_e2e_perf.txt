     with strictPort + `@` alias to project src.

     Edit Memo... -> RegionMemoDialog with codeName.

2. Konva perf baseline
   - tools/perf/perf-page.html + src/perf.ts: standalone
     Konva harness. 256 / 1024 / 2048 synthetic mid-grey PNGs
     (built via in-page Canvas2D, mirroring the production
     data: URL + Image-onload path), N=200 iterations each
     of bbox draw + polygon push via performance.now(). Emits
     JSON to #results + console.log.
   - tools/perf/perf-bench.spec.ts: Playwright driver;
     navigates to fixture perf page, captures #results,
     writes tools/perf/results.json, asserts reasonable
     bounds (0.5ms < avg < 200ms per draw).

     research writeup with methodology, expected thresholds

     comparison to v0.3 with motivation + next steps.
