# FU3 - Unblock the release matrix

The release matrix for github.com/mabo-du/lens is still failing in 5-6 sec
across all 4 platforms with no logs, even after round-73 tauri-action SHA
pin + job permissions + repo workflow permissions. The most likely
remaining cause is an org-level third-party-action blocklist on Settings
-> Actions -> General at the mabo-du org level.

This needs admin:org scope. From your LOCAL terminal:

  gh auth refresh -h github.com -s admin:org

After that:

  gh api orgs/mabo-du/actions/permissions/workflow
  gh api orgs/mabo-du/actions/permissions
  gh api orgs/mabo-du 2>&1 | head -30

If policy needs lifting:

  gh api orgs/mabo-du/actions/permissions/workflow --method PUT \
    -F default_workflow_permissions=write

Then re-trigger the release workflow:
  gh workflow run release.yml --ref main
(LENS is at commit 863ddb9 / tag v0.2-rc-complete as of round-77)
